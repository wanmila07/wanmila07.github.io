Developed during an industry internship at Gagasan Consultant, Kuantan, this system replaced a fully manual project record-keeping process with a secure, bilingual web application.

**Key features:**
- CRUD operations with role-based authentication
- SQL injection and XSS protection
- Analytics dashboard with visual charts (projects by year, type, and status)
- Bilingual interface (English and Bahasa Malaysia)
- Responsive design with cross-browser compatibility

**Technical stack:** MySQL, PHP, Apache/MAMP, HTML, CSS, Figma

**Deliverables:** Full technical documentation including requirements analysis, installation guide, maintenance guide, and test report.

This project demonstrates end-to-end system development capability — from requirements gathering and UI design through to deployment and documentation.
